# סדנאות תיפוף

## סקירה
דף נחיתה לסדנאות תיפוף - team building, אירועים, קבוצות.

## Tech Stack
- Next.js 15 + TypeScript + Tailwind CSS 4

## סגנון
- חם, אנרגטי, צבעים אדמתיים (כתום, חום, אדום כהה)
- Heebo font, RTL מלא

## מבנה
1. Hero - כותרת + תמונה דינמית
2. מה זה סדנת תיפוף?
3. למי מתאים? (חברות, אירועים, בתי ספר, קהילות)
4. גלריה (תמונות אותנטיות + Gemini)
5. המלצות
6. יצירת קשר / הזמנת סדנה

## תמונות

- חובה: ייצר עם Gemini. אסור placeholders.
